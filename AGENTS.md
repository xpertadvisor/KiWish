# KiWish 项目 - 智能代理指导文档

## 项目概述

KiWish 是一个专门的网站项目，展示一种名为 "KiWish Detox" 的健康排毒饮料产品。该产品定位为"顽固脂肪减少和清洁燃烧配方"，是一种清爽的植物基混合饮料，旨在从内到外地温和清洁身体。它富含抗氧化水果、天然纤维和植物提取物，有助于支持消化、减少腹胀并焕发光泽，且不含刺激性成分或人工添加剂。

## 项目架构

- **前端技术栈**: HTML, CSS, JavaScript
- **样式框架**: Tailwind CSS
- **图标库**: Lucide 图标
- **字体**: Google Fonts (Inter, Playfair Display)
- **响应式设计**: 适用于所有设备尺寸
- **主要文件**: `Source/Index.html` - 网站的主页面

## 产品特性

### 核心功效
- **排毒**: 温和清洁肠道和肝脏，清除毒素
- **新陈代谢**: 提升新陈代谢，即使在休息时也能燃烧脂肪
- **消化**: 改善消化，减少腹胀，让肠道感觉更轻松
- **体重管理**: 阻止脂肪储存，将现有脂肪转化为能量
- **胆固醇**: 降低胆固醇，维持健康的血脂水平
- **血糖**: 调节血糖水平
- **肠道健康**: 增加有益菌群，支持消化系统健康
- **营养吸收**: 提高生物利用度和营养吸收

### 三大步骤过程
1. **清洁** - 当身体系统超载时，脂肪燃烧会减慢。KiWish 温和清洁你的肠道和肝脏，清除毒素和废物，重置你的身体。
2. **提升新陈代谢和燃烧脂肪** - 阻止脂肪储存并将现有脂肪转化为能量，让身体自然融化顽固脂肪，无需剧烈锻炼。
3. **收腹** - 改善消化和腹胀。你的肠道感觉更轻松，腰围开始显现差异。

### 主要成分

#### 5 种主要绿色成分
1. **新西兰猕猴桃** - 营养价值更高，生长环境更纯净，品质一致，口感和质地更佳
2. **叶绿素** - 植物中的天然绿色素
3. **青苹果** - 富含抗氧化剂和纤维
4. **黄瓜** - 保湿且低热量的蔬菜
5. **小麦草粉** - 营养丰富的超级食品

#### 专利成分
- **KiOnutrime-Cs®** (比利时) - 多酶复合物，能结合至少800倍自身重量的脂肪，EFSA批准的健康声明
- **Sunfiber®** (印度) - 瓜尔豆纤维，降低胆固醇和血糖水平，增加肠道有益菌群
- **DigeZyme®** (美国) - 植物壳聚糖，帮助降低脂肪储存，防止体重和脂肪增加

## 使用方法

1. 每日取一袋
2. 与150毫升水混合
3. 可随时服用（最好在餐后或睡前）
4. 粉末易溶于水，轻轻搅拌即可
